# 03 - DataFrames

## Topics
- Creating DataFrames from lists, CSV, JSON, Parquet
- Schema definition
- select, filter, withColumn, drop
- Joins (inner, left, right, outer)
- Handling nulls
- User Defined Functions (UDFs)

## Key Mental Model
A DataFrame is like a table in a database or a pandas DataFrame —
rows and columns with a schema. But it is distributed across
multiple machines and processed in parallel.
