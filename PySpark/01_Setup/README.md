# 01 - Setup & Installation

## Goal
Get PySpark running on your Windows machine and verify everything works.

## Steps
1. Install Java
2. Install PySpark
3. Configure environment variables
4. Test with a Hello World script

## Notes
- Java must be installed BEFORE PySpark
- Use Java 8 or Java 11 (Java 17+ may cause issues)
- PySpark version should match your use case (3.x recommended)
