# 05 - Data Engineering

## Topics
- ETL (Extract, Transform, Load) pipeline pattern
- Reading and writing file formats (CSV, JSON, Parquet, ORC)
- Partitioning and bucketing
- Data quality checks
- Handling large datasets efficiently

## Key Mental Model
Data Engineering = moving data from source to destination
reliably, at scale, with transformations in between.
Parquet is your best friend — columnar, compressed, fast.
